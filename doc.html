How We Did It

When considering what would be most helpful in future travel planning, we searched high and low for the best free APIs out there. After an exhaustive search, we found the TravelBriefing API. This API has been hugely helpful in accessing all of the information we need. It offers a lot of information, without those pesky call limits.


We also utilized the Google Maps API to provide location data, along with Jquery to assist in our JavaScript and JQuery UI to help style our auto-complete.


In addition to the above, we also utilized Bulma as our CDN. Originally, we were attracted ot what appeared to be an easy to use design. We were wrong. While we were able to make things work in the end, Bulma definitely did not help as much as it hindered.
